#  My Reference Library

A curated collection of definitive reference manuals and textbooks covering many things of Computer Science, and core programming paradigms. 
These texts skip beginner tutorials and focus on architecture, idiomatic design, and deep technical mastery.

## Book Collection

Click on any title below to open and read the PDF directly in your browser.

### Systems & Core Languages
* **C++:** [The C++ Programming Language (4th Edition) - Bjarne Stroustrup](C++/The%20C%2B%2B%20Programming%20Language%20%5B4th%20Edition%5D%20-%20Bjarne%20Stroustrup.pdf)
  > The definitive reference manual written by the creator of the language.
* **Rust:** [Programming Rust (2nd Edition) - Jim Blandy, et al.](Rust/Programming%20Rust%202nd%20Edition-1.pdf)
  > A rigorous deep-dive into safe concurrency, memory layout, and zero-cost abstractions.
* **Go:** [The Go Programming Language - Alan Donovan & Brian Kernighan](Go/The%20Go%20Programming%20Language%20-%20Donovan,%20Alan%20A.%20A.%20_%20Kernigha_6127.pdf)
  > The authoritative text on Go's architecture and concurrent goroutine model.
* **Python:** [Fluent Python (2nd Edition) - Luciano Ramalho](Python/luciano-ramalho-fluent-python_-clear-concise-and-effective-programming-oreilly-media-2022.pdf)
  > The ultimate encyclopedia of Python's underlying data model and execution engine.

### Shell & Environment (Bash)
* **Standard Reference:** [bash Cookbook (O'Reilly)](Bash/O'Reilly%20bash%20CookBook.pdf)
  > Practical, idiomatic solutions for shell scripting and automation.
* **Advanced Mastery:** [Advanced Bash-Scripting Guide (ABS)](Bash/abs-guide.pdf)
  > The legendary, exhaustive manual for the darkest corners of shell scripting.

### Computer Science Foundations
* **Operating Systems:** [Operating System Concepts (10th Edition) - Abraham Silberschatz](Operating-Systems/Abraham-Silberschatz-Operating-System-Concepts-10th-2018.pdf)
  > The industry-standard textbook for understanding processes, memory management, and file systems.

### Compilers
* **Dragon Book:** [Compilers: Principles, Techniques, and Tools (2nd Edition) - Aho, Lam, Sethi, Ullman](Compilers/Alfred-V.-Aho-Monica-S.-Lam-Ravi-Sethi-Jeffrey-D.-Ullman-Compilers-Principles-Techniques-and-Tools-Pearson_Addison-Wesley-2007.pdf)
  > The classic "Dragon Book" — the definitive reference on compiler design and construction.
* **Engineering a Compiler:** [Engineering a Compiler (2nd Edition) - Cooper & Torczon](Compilers/Engineering%20a%20Compiler%202nd%20ed%20-%20K.%20Cooper%2C%20L.%20Torczon%20%28Morgan%20Kaufman%2C%202012%29%20BBS%20bbbb.pdf)
  > A modern, rigorous approach to compilation, from parsing to code generation.

### Networking
* **Computer Networks:** [Computer Networks (6th Edition) - Tanenbaum, Feamster, Wetherall](Networking/Andrew%20S.%20Tanenbaum%2C%20Nick%20Feamster%2C%20David%20J.%20Wetherall%20-%20Computer%20Networks-Pearson%20%282021%29.pdf)
  > The authoritative textbook covering protocols, architecture, and network design.

### Kotlin
* **Language Reference:** [Kotlin Reference](Kotlin/kotlin-reference.pdf)
  > The official language reference and specification.
* **Android Development:** [Learn Kotlin for Android Development - Apress](Kotlin/Apress.Learn.Kotlin.for.Android.Development..www.EBooksWorld.ir.pdf)
  > Practical guide to building Android applications with Kotlin.
